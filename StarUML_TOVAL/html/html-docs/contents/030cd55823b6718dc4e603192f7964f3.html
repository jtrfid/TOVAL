Description

This class manages a number of points and places them in a circular way according to the minimum distance between any two points. <br> For up to 4 points the class places the points in a trivial way:<br> <br> 1: o<br> 2: o o<br> 3: o<br> o o<br> 4: o o<br> o o<br> <br> For 5 or more points, the class places on point at the origin and then introduces one or more circles around this central point and places them on the circular line(s). @author ts552
