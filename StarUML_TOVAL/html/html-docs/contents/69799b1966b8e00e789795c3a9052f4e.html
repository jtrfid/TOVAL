Description

Interface for value generators.<br> A value generator produces values of a specific type and returns it by the method {@link #getNextValue()}. Value generation is only permitted if the generator is in a valid state.<br> Once entered a valid state, the generator cannot enter an invalid state again.<br> If value generations are requested in invalid state, Exceptions of type {@link ValueGenerationException} are thrown. @param <E>
