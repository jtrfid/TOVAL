makeCompactGrid

UMLOperation
Untitled :: JavaReverse :: de :: invation :: code :: toval :: graphic :: util :: SpringUtilities :: makeCompactGrid

Description

Aligns the first <code>rows</code> <code>cols</code> components of <code>parent</code> in a grid. Each component in a column is as wide as the maximum preferred width of the components in that column; height is similarly determined for each row. The parent is made just big enough to fit them all. @param rows number of rows @param cols number of columns @param initialX x location to start the grid at @param initialY y location to start the grid at @param xPad x padding between cells @param yPad y padding between cells
