Description

<p> Enum to represent the different program categories. The categories have three different types: </p> <ol> <li><i>Main Categories</i> consists of those categories that every conforming desktop environment must support.</li> <li><i>Additional Categories</i> provides categories that can be used to provide more fine grained information about the application.</li> <li><i>Reserved Categories</i> containes categories that have a desktop-specific meaning.</li> </ol> @version 1.0 @author Adrian Lange lange@iig.uni-freiburg.de
