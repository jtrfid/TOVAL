Description

This class can be used to request a file/directory with the help of a JFileChooserDialog.<br> It allows to customize the accepted files/directories by overriding the method {@link #isValid(File)}.<br> The user will be shown a JFileChooserDialog until he hits cancel or selects a valid file. @author Thomas Stocker
