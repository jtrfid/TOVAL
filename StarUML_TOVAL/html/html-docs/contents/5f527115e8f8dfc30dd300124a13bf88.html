Description

Basic panel for 2-dimensional diagrams.<br> Values for different coordinate axes are kept separately. This gives users the chance to define different kinds of diagrams by changing the way values are depicted inside the diagram itself. The standard behavior defined in this class is that the coordinates of depicted points directly correspond to the given value sets. Diagram panels can basically operate in two different modes: zero-based or not zero-based. In zero-based mode the coordinate axes always start at zero and end near the highest value for the corresponding dimensions. In non-zero mode the axes start near the smallest values. @author Thomas Stocker
