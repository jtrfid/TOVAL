This way just constructors with the required parameters are needed and an arbitrary amount of vararg attributes can be used. To set the attributes just one object call is needed, which reduces the code. </p> <p> The following example creates a {@link LinuxProgramStarter}, which uses this interface: </p> <pre> LinuxProgramStarter starter = new LinuxProgramStarter.Builder("fn", LinuxProgramStarter.ProgramStarterType.LINK, "Name", "bin/bash program.sh").addMimeType("text/xml").build(); </pre> @see StringBuilder @version 1.0.2 @since 1.0.2 @author Adrian Lange lange@iig.uni-freiburg.de @param <T> Type to create the builder for
