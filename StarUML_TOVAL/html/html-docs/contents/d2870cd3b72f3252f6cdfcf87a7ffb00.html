getTruthTable

UMLOperation
Untitled :: JavaReverse :: de :: invation :: code :: toval :: math :: MathUtils :: getTruthTable

Description

Constucts a truth-table for the given column-number. <br> The number of table rows is 2^columnCount.<br> A truth-table with three columns looks like this:<br> (0=<tt>false</tt>, 1=<tt>true</tt>)<br> <br> <ul> <li>0 0 0</li> <li>0 0 1</li> <li>0 1 0</li> <li>0 1 1</li> <li>1 0 0</li> <li>1 0 1</li> <li>1 1 0</li> <li>1 1 1</li> </ul> <br> @param colCount The number of columns. @return The constructed truth-table
