Description

Matrix representation where any type can be used for indexing cols and rows.<br> Internally values are put in a 2-dimensional list, represented by an ArrayList of ArrayLists. Indexing values are mapped onto integers that serve as indexes for retrieving the appropriate values out of the list. @param <E> Indexing type for rows and cols @param <T> Value type for matrix entries @author Thomas Stocker
