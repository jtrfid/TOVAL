Description

Divides the given object-list using the boundaries in <code>cuts</code>. <br> Cuts are interpreted in an inclusive way, which means that a single cut at position i divides the given list in 0...i-1 + i...n<br> This method deals with both cut positions including and excluding start and end-indexes<br> @param list List to divide @param cuts Cut positions for divide operations @return A list of sublists of <code>list</code> according to the given cut positions @see #divideList(List, Integer...)
