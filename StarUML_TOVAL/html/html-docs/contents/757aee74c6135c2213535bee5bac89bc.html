Description

Returns a {@link LinkedHashSet} of all classes in a specified package. Since the result is returned as a {@link LinkedHashSet}, there won't be any duplicates. @param packageName Package name. @param recursive <code>true</code> if subpackages should be considered, too. @return {@link LinkedHashSet} of classes in a specified package. @throws ReflectionException
