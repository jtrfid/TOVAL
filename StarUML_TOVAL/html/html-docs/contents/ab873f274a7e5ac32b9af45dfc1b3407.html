Description

This class can be used to determine all different overlapping groups for a set of time intervals.<br> An interval with borders a,b with a<b is said to overlap another interval with borders c,d with c<d, iff<br> c lies within [a;b). The algorithm works in an online fashion and reports detected overlap groups to interested listeners (registration via {@link AbstractIntervalOverlap#registerListener(OverlapListener)} is required) as they occur.<br> Please use the method {@link AbstractIntervalOverlap#reportTimeInterval(long, long)} to report new intervals. @author Thomas Stocker
