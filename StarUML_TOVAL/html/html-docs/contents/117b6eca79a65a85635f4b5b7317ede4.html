Description

Class for modeling multisets.<br> A multiset can contain contain the same element multiple times.<br> Given a set S, a multiset can be defined as a mapping m : S -> N.<br> The number m(s) is called the multiplicity of s in m.<br> <br> Objects are considered equal if they have the same hash.<br> @author Thomas Stocker @param <O>
