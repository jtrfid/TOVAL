The Windows Registry is the configuration catalogue from Microsoft Windows. </p> <p> It is divided into different <i>hives</i> for different purposes (<code>HKEYCLASSESROOT</code>, <code>HKEYCURRENTUSER</code>, <code>HKEYLOCALMACHINE</code>, etc.), which contain hierarchically structured <i>keys</i>. Each key can contain <i>values</i>, which have a name and a content. Every key contains a default value, usually indicated by an empty name ({@value #DEFAULTKEYNAME}). </p> <p> This class contains methods to read and write keys and values in different hives. </p> @version 1.0 @author Adrian Lange lange@iig.uni-freiburg.de
