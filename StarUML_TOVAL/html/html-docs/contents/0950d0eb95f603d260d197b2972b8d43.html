Description

Class for managing information about ticks adjusted to i.e. axes or lines.<br> Ticks have major and minor ticks which can have different sizes (lengths). The specified tickRange applies to minor ticks; major tick ranges are defined as multiples of minor ticks with the help of an adjustable multiplicator. Basically there are two operation modes: zero-based or not.<br> In zero-based mode ticks always start at 0 and end near the maximum value of the corresponding value range. Otherwise ticks start at the minimum value.
